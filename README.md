# 🐍 Console Maze Game (コンソール迷路ゲーム)

このプロジェクトは、Pythonを使用して作成された、コマンドライン上で遊べるシンプルな迷路脱出ゲームです。ランダムに生成される迷路を操作コマンド（w/a/s/d）で探索し、ゴールを目指します。

---

## ✨ ゲームの概要

* **目的:** プレイヤー（`P`）を操作して、壁（`█`）を避けながらゴール（`G`）に到達することを目指します。
* **ステージ:** ゴールに到達すると自動的に新しいステージ（新しい迷路）が生成されます。
* **プラットフォーム:** ターミナルまたはコマンドプロンプト上で動作します。

---

## 🛠️ 技術スタック

* **言語:** Python 3.x
* **ライブラリ:** 標準ライブラリ (`random`, `os`, `sys`) のみ使用。
* **迷路生成アルゴリズム:** バックトラック法（深さ優先探索）に基づいています。

---

## 💻 使い方

### 1. ファイルの準備

コードを `maze_game.py` などのファイル名で保存します。

### 2. 実行

コマンドライン（ターミナルやコマンドプロンプト）で以下のコマンドを実行します。

```bash
python maze_game.py
